<p>탐레프는 불순 전자들이 자주 이동하는 경로를 찾기 위해 수시로 전자 현미경의 실적들을 조회한다. 전자가 이동하거나 실적을 조회하는 쿼리 <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D45E TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>q</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$q$</span></mjx-container>개를 처리하는 프로그램을 작성하여 보자.</p>
